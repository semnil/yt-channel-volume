# プライバシーポリシー — YT Channel Volume

最終更新日: 2026-04-11

## 概要

YT Channel Volume は、YouTube チャンネルごとに音量を記憶・自動適用する Chrome 拡張機能です。本ポリシーでは、拡張機能が扱うデータの種類、利用目的、保存先について説明します。

## 収集するデータと目的

### チャンネル音量設定

- **対象**: YouTube チャンネル ID、表示名、チャンネル URL、および設定したボリューム（ゲイン）値
- **目的**: 保存済みチャンネルの動画を開いた時に、設定した音量を自動適用
- **保存先**: デバイス上の `chrome.storage.local` にローカル保存。外部サーバーには一切送信されません

### 拡張機能の設定

- **対象**: ターゲット LUFS 値、表示単位（% または dB）、ゲイン表示のオン・オフ
- **目的**: ユーザーの好みに応じた動作のカスタマイズ
- **保存先**: デバイス上の `chrome.storage.local` にローカル保存

### YouTube 動画メタデータ（読み取りのみ）

- **対象**: Content Loudness (`loudnessDb`)、`isLiveContent`、`isLive`、`channelId`、`videoId` — YouTube のプレイヤーレスポンスから取得
- **目的**: ラウドネス情報の表示および推奨ボリュームの算出。ライブ配信か通常動画かの判定
- **保存**: 保存されません。動画ページを開いている間のみメモリに保持し、ページ遷移時に破棄されます

## 収集しないデータ

- 閲覧履歴、アナリティクス、テレメトリは**収集しません**
- YouTube やその他のサイトでの閲覧ページは**追跡しません**
- 拡張機能の開発者はユーザーのデータを**受信・保存・アクセスしません**
- データの販売、第三者との共有、広告目的での利用は**一切行いません**

## データの送信先

ありません。この拡張機能は**外部へのネットワークリクエストを一切行いません**。すべてのデータはデバイス上に留まります。

## データの保存とセキュリティ

- すべての設定は `chrome.storage.local` に保存され、この拡張機能のみがアクセス可能です
- デバイス間での同期やクラウドへの保存は行いません
- 拡張機能をアンインストールすると、ローカルに保存されたすべてのデータが削除されます

## 権限

| 権限 | 理由 |
|---|---|
| **storage** | チャンネル音量設定とユーザー設定をローカルに保存 |
| **activeTab** | 現在の YouTube タブにアクセスし、チャンネルの検出、Content Loudness の読み取り、音量調整を適用 |
| **host_permissions** (`youtube.com`) | YouTube ページにコンテンツスクリプトを注入し、Web Audio API で音量を制御 |

## リモートコード

この拡張機能は**リモートコードを使用しません**。すべての JavaScript は拡張機能パッケージ内にローカルで同梱されています。`page-bridge.js` はページのメインワールド (`"world": "MAIN"`) で実行されますが、これは YouTube のプレイヤーレスポンスデータを読み取るためのローカルコードであり、リモートから取得されたものではありません。

## 単一用途

この拡張機能の用途は1つです: YouTube の Content Loudness メタデータと Web Audio API を使用して、**チャンネルごとの音量設定を調整・記憶する**こと。

## 第三者依存

なし。外部ライブラリ、SDK、CDN、アナリティクスツールは一切使用していません。

## ポリシーの変更

変更がある場合は更新日を改訂の上、本ページに掲載します。変更後も拡張機能を継続使用した場合、改訂後のポリシーに同意したものとみなします。
